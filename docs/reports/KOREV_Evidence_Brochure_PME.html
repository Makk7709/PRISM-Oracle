1. Architecture et Différenciation Massive

KOREV Evidence n'est pas un chatbot. C'est un système multi-agents avec validation intégrée, conçu pour les métiers où une erreur a des conséquences.

Dimension Ce que propose KOREV Evidence Mécanisme / Preuve
Orchestration agents 9+ agents spécialisés, hiérarchie, policy déterministe Agent 0, délégation, multi-domaine
Validation / responsabilité Consensus multi-LLM, fail-closed, audit trail Quorum 2/3, refus explicite
Sécurité / Conformité Auth Argon2id, rédaction logs, CSRF, PII, rate limiting Testé OWASP, conforme RGPD
Traçabilité Evidence pack, bundle d'audit, correlation IDs Citation PMID/DOI/loi
Limitation / intégrité Agents expriment leurs limites, refuse de simuler Journalisation, modes prod bloqués
Ce qui change tout

KOREV Evidence se distingue par une confiance systémique : chaque réponse citée, traçabilité irréprochable, et barrière fail-closed qui garantit zéro hallucination non détectée — là où les IA génériques se contentent d'affirmer, Evidence préfère refuser. Avec près de 4 000 cas de tests automatisés (snapshot probatoire 28 avril 2026) et 8 invariants sur les process critiques, la sécurité et la fiabilité du système dépassent les standards marché.

5. Les garanties Evidence

Chaque fonctionnalité de confiance est implémentée, testée et auditable dans le code source.

Garantie Ce que cela signifie Preuve technique
Chaque affirmation sourcée PMID, DOI, article de loi, label FDA — jamais d'hallucination non tracée Evidence Pack, citations vérifiées contre index local
Consensus multi-LLM Les réponses critiques sont validées par vote entre plusieurs modèles d'IA Quorum 2/3, 3 arbitres indépendants
Fail-closed Le système refuse plutôt que d'inventer — zéro faux positif silencieux 8 invariants P0, ~3 956 cas de tests collectés (snapshot 28 avril 2026)
Débat contradictoire 3 rounds de confrontation pour les affirmations complexes Collaborative debate engine
Audit trail complet Chaque réponse traçable avec identifiant unique Correlation IDs, bundle d'audit déterministe
Limites assumées Chaque agent connaît ses frontières et les exprime Classification criticité, refus documenté
Privacy-First Architecture
  • Anonymisation PII dans tous les journaux d'audit (emails, téléphones, IBAN, numéros de sécurité sociale)
  • Portabilité native : export JSON, PDF, backup ZIP de toutes vos données
  • Suppression à la demande des conversations et mémoires
  • Option déploiement souverain avec LLM local (Mistral/Ollama) — aucune donnée ne quitte votre infrastructure
  • Transport sécurisé TLS/HTTPS, authentification Argon2id
Prêt pour l'AI Act

L'AI Act européen impose dès août 2027 des obligations de traçabilité et d'auditabilité pour les systèmes IA à haut risque. KOREV Evidence intègre nativement ces exigences : audit trail, documentation des décisions, gestion des risques et supervision humaine.
